Maintenance Hub

A client project shaped around residence maintenance tickets, campus-based administration, real-time updates, and secure role-based access.

About Project

Maintenance Hub is a Flutter and Firebase application I built for a client to manage residence maintenance requests across multiple campuses. The system supports structured ticket reporting, issue categorisation, administrator workflows, caretaker assignment, and real-time communication between students and support staff.

The updated requirements expanded the project from a simpler maintenance request tool into a larger operational platform with multi-level administration, residence-specific permissions, analytics controls, notifications, secure file handling, and ticket-linked messaging.
